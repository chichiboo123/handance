한글춤 설명서 📚

한글을 사랑하고 아끼는 마음으로 만들었습니다 ❤️

✏️ 텍스트 입력

위쪽 상자에 한글을 입력하면 예쁜 글자가 화면에 나타나요!

🎨 색상 바꾸기

배경 버튼을 누르면 뒤 배경색을 바꿀 수 있어요.

글자 버튼을 누르면 글자 색을 바꿀 수 있어요.

🖼️ 배경 파일

사진이나 동영상을 배경으로 넣을 수 있어요!

📝 폰트 선택

21가지 예쁜 한글 글꼴 중에서 골라보세요.

🎮 슬라이더 조절

강도: 글자가 얼마나 움직일지 정해요.

방향: 가로/세로로 움직이는 방향을 정해요.

크기: 글자 크기를 키우거나 줄여요.

간격: 글자 사이 간격을 조절해요.

↩️ 실행 취소/다시 실행/초기화

화살표 버튼으로 이전 상태로 돌아가거나 다시 실행할 수 있어요.

새로고침 버튼을 누르면 모든 슬라이드를 지우고 처음 상태('한글춤' 1개)로 돌아가요.

📹 녹화하기

GIF 또는 WebM 형식을 선택하세요.

녹화 시작 버튼을 누르면 녹화가 시작돼요!

다시 버튼을 누르면 멈추고 자동으로 다운로드돼요.

🎞️ 슬라이드 관리

슬라이드 추가: 새로운 화면을 만들어요.

삭제: 현재 슬라이드를 지워요. (마지막 1개는 지울 수 없어요)

이전/다음: 슬라이드를 넘겨가며 볼 수 있어요.

🌍 언어 변경

KOR/ENG 버튼을 누르면 한국어와 영어로 바꿀 수 있어요!

🌙 테마 바꾸기

달/해 버튼을 누르면 어두운 모드와 밝은 모드를 바꿀 수 있어요!
